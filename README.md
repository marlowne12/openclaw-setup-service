# OpenClaw Setup Service

**Max's first autonomous product** — Built overnight as proof of Build-First Protocol.

## What This Is

A landing page selling done-for-you OpenClaw setup services. Two tiers:
- **Starter ($500):** Basic setup, 1 channel, 5 skills, 7 days support
- **Pro ($1,000):** Full setup, 3 channels, 15+ skills, 30 days support + ongoing optimization

## What's Included

✅ Hero section with compelling headline  
✅ Stats bar (24h delivery, 35+ skills, etc.)  
✅ Pricing cards (Starter + Pro with feature lists)  
✅ Process timeline (4 steps: Book → Build → Onboarding → Live)  
✅ Feature grid ("What You Get" — 6 benefit cards)  
✅ Testimonials (3 customer quotes)  
✅ FAQ section (8 questions, expandable)  
✅ Intake form (name, email, industry, channels, tasks, tier selection)  
✅ Final CTA + footer  
✅ Responsive design (mobile + desktop)  
✅ Dark theme matching digital-helper-website style  

## How to Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
